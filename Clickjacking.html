Red Team Assessment – BSSA

Pengujian ini bertujuan untuk mengidentifikasi apakah halaman web dapat dimuat di dalam <iframe>. Jika halaman berhasil ditampilkan, maka target berpotensi rentan terhadap Clickjacking.

Mitigasi:
Batasi pemuatan halaman ke dalam iframe menggunakan HTTP Security Header. X-Frame-Options: DENY X-Frame-Options: SAMEORIGIN Content-Security-Policy: frame-ancestors 'none'; Referensi:
https://cheatsheetseries.owasp.org/cheatsheets/Clickjacking_Defense_Cheat_Sheet.html
https://www.rfc-editor.org/rfc/rfc7034
