Pengujian untuk memastikan apakah halaman web dapat dimuat di dalam <iframe>.
Iframe diblokir oleh target.
Target tidak rentan terhadap Clickjacking.
Mitigasi: Batasi pemuatan halaman ke dalam iframe menggunakan HTTP Security Header. X-Frame-Options: DENY X-Frame-Options: SAMEORIGIN Content-Security-Policy: frame-ancestors 'none';
