Clickjacking Testing Tool Pentest

Clickjacking Testing Tool digunakan untuk menguji apakah suatu website dapat dimuat di dalam elemen <iframe>. Jika halaman target berhasil ditampilkan, maka website tersebut berpotensi rentan terhadap serangan Clickjacking.

Rekomendasi Mitigasi:
Terapkan HTTP Response Header untuk mencegah halaman dimuat di dalam iframe dari domain yang tidak sah. X-Frame-Options: DENY X-Frame-Options: SAMEORIGIN Content-Security-Policy: frame-ancestors 'none'; Referensi:
OWASP Clickjacking Defense Cheat Sheet
RFC 7034 – X-Frame-Options
⚠️ Digunakan hanya untuk keperluan pengujian keamanan yang sah dan berizin.
